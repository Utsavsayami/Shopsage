Yo repo ma 2 ota folder xa.
ShopSage-Ecommerce (backend ).
shopsage (Flutter ).

Pahila Git, Node.js, MongoDB ra Flutter install hunu parcha.

Project clone garna:
git clone https://github.com/Utsavsayami/Shopsage.git
cd Shopsage

Already clone gareko cha bhane latest code lina:
git pull

Backend run garna:

cd ShopSage-Ecommerce
npm install
npm run dev

Backend terminal close garnu vayena. MongoDB pani run bhairako hunu parcha.

Backend URL:
http://localhost:5000

Flutter run garna side by side aarko terminal kholnu parcha:

cd Shopsage/shopsage
flutter pub get
flutter run
